===========*Installation Template Instruction*==========
*Please make following steps: 
*1. Go to your FTP (root/templates) (where root - is main derectory for Boonex Dolphin script) 
*2. Put 'tmpl_cha' to 'root/templates' folder
*3. Go to Admin Panel->Settings->Advanced Settings then choose "Chamel" skin as default one on your site.
*4. Click "Clean All" in Admin Panel->Dashboard (to clean cache) 

===============*Fix the Forum Page*=====================
Copy and place the folders(cha704) from the forum folder into the path  (\modules\boonex\forum\layout\)

=================*Fix the Poll Box*=====================
Copy and place the folders(tmpl_cha704) from the poll folder into the path  (\modules\boonex\poll\templates\)
